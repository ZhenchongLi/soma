# v0.3 Test Contract â€” LFE DSL Compiler

This document maps each behavioural property of the v0.3 LFE DSL compiler layer
to the test that proves it. It is the companion to
[v0.2-test-contract.md](v0.2-test-contract.md) and serves as the barrier that
prevents future work from accidentally treating the DSL as a runtime component.

## Compile-only contract

The `soma_lfe` app is a compile-only layer. It must not start processes, emit
runtime events, call tools, or carry a runtime dependency on `soma_runtime`.

| Property | Test module | Test name |
|----------|-------------|-----------|
| Compiler has no runtime dependency at compile time or at runtime | `soma_lfe_tests` | `test_soma_lfe_does_not_depend_on_soma_runtime` |

## Compiler output

Compilation returns either `{ok, #{run => #{steps => Steps}}}` or
`{error, [Diagnostic]}`. The `steps` value is the exact list of maps accepted
by `soma_agent_session:start_run/2` today.

| Property | Test module | Test name |
|----------|-------------|-----------|
| A valid single-step DSL compiles to the expected step map | `soma_lfe_compile_tests` | `test_compile_single_step` |
| The demo (`file_read â†’ echo â†’ file_write`) compiles correctly | `soma_lfe_compile_tests` | `test_compile_demo_three_steps` |
| Bare `from_step` compiles to `#{from_step => StepId}` | `soma_lfe_compile_tests` | `test_compile_bare_from_step` |
| Field-level `from_step` compiles to `Key => {from_step, StepId}` | `soma_lfe_compile_tests` | `test_compile_field_from_step` |
| `timeout_ms` present compiles to `timeout_ms => N` | `soma_lfe_compile_tests` | `test_compile_with_timeout` |
| `timeout_ms` absent is omitted from the step map | `soma_lfe_compile_tests` | `test_compile_without_timeout` |
| String values in args become binaries | `soma_lfe_compile_tests` | `test_compile_string_becomes_binary` |

## Validation and diagnostics

Diagnostics are structured maps with a stable `code` field. Compilation returns
`{error, Diagnostics}` for any invalid input â€” never a partial step list.

| Property | Test module | Test name |
|----------|-------------|-----------|
| Duplicate step ids return `duplicate_step_id` | `soma_lfe_validation_tests` | `test_duplicate_step_id_returns_diagnostic` |
| Unknown `from_step` reference returns `invalid_from_step` | `soma_lfe_validation_tests` | `test_unknown_from_step_returns_diagnostic` |
| Forward `from_step` reference returns `invalid_from_step` | `soma_lfe_validation_tests` | `test_forward_from_step_returns_diagnostic` |
| Invalid `timeout_ms` value returns `invalid_timeout` | `soma_lfe_validation_tests` | `test_invalid_timeout_returns_diagnostic` |
| Unknown DSL forms return `unknown_form` | `soma_lfe_validation_tests` | `test_unknown_form_returns_diagnostic` |
| Compile failure does not start a run or emit runtime events | `soma_lfe_validation_tests` | `test_invalid_dsl_does_not_start_run` |
